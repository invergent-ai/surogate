# Configuration

Surogate is driven by a YAML config file.

## Start from an example

- SFT examples live in `examples/sft/`
- Pretraining examples live in `examples/pt/`

Run via CLI:

```bash
surogate sft path/to/config.yaml
# or
surogate pt path/to/config.yaml
```
